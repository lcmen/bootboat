Breadcrumb

Default (with slashes)

  1. Home
  2. / Components
  3. / Breadcrumb
<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li class="breadcrumb-item"><a href="#">Home</a></li>
    <li class="breadcrumb-separator">
      <span>/</span>
      <a href="#" class="breadcrumb-item">Components</a>
    </li>
    <li class="breadcrumb-separator">
      <span>/</span>
      <span class="breadcrumb-current">Breadcrumb</span>
    </li>
  </ol>
</nav>

With chevrons

  1. Home
  2. Components
  3. Breadcrumb

  <svg class="h-4 w-4">...</svg>

</li>
