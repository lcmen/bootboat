Bootboat

Sailboat UI component styles as reusable CSS classes, built on Tailwind CSS v4. Write btn btn-primary instead of long utility strings.

Get Started

Download bootboat.css and add it to your Tailwind CSS v4 project. Requires @tailwindcss/forms plugin for form elements.

Download Latest Release View on GitHub
npm install @tailwindcss/forms
@plugin "@tailwindcss/forms" { strategy: base }
@import "./bootboat.css";
